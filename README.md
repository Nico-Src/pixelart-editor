# Pixelart Editor
![Current State](https://raw.githubusercontent.com/Nico-Src/pixelart-editor/refs/heads/main/public/img/screen.png)

## Features
- Multilayer Canvas System
- Axis Snapping
- Saving/Loading to Disk (.pxl file)
- Exporting Image
